# Contact Management System

## Description
This is a simple Contact Management System developed using Python. 
It allows users to add contacts, view all contacts, search for a contact, and exit the program.

## Features
- Add new contact
- View all contacts
- Search contact by name
- Exit program

## Technologies Used
- Python
- Dictionary Data Structure
- While Loop & Conditional Statements

## How It Works
1. User selects an option from menu
2. Add contact (Name & Phone Number)
3. View all contacts
4. Search for a contact
5. Exit program
